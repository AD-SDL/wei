================
Deployment Guide
================

Everything you need to know about installing, deploying, and/or configuring WEI, WEI-powered Workcells, and WEI-compatible modules.

We recommend Docker-based deployments for most users, as it provides superior reliability, stability, portability, and security, and makes managing more complex setups easier. However, if you have specific requirements or are interested in deploying WEI on bare metal, we have you covered.
